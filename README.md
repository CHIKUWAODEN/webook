## Install

RubyGems では公開していないので、次のような感じで手動でインストールしてください。

    % cd path/to/tmp
    % git clone https://github.com/ykanda/webook.git
    % cd webook
    % rake build
    % gem install --local ./pkg/webook-0.0.1.gem
